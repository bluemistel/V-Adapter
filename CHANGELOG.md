# 更新履歴 / Changelog

本プロジェクトのすべての注目すべき変更を記録します。
形式は [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/)、バージョニングは [Semantic Versioning](https://semver.org/lang/ja/) に準拠します。

## [0.0.1] - 2026-06-23 (α)

初回リリース（α版）。

### 追加
- 標準マクロ「音声の再生（Ctrl+Space）」「音声の保存（Ctrl+E）」を同梱
- 対象アプリのテンプレート（VOICEVOX / A.I.VOICE2 / CeVIO AI / VOICEPEAK / VoiSona Talk）を同梱
- 1つのマクロに対象アプリごとのスクリプト（命令列）を保持する構造
- 実行時のスクリプト自動選択（前面アプリ判定）と「送信先」固定
- マクロ命令: 待機 / クリック / キー送信 / ウィンドウ表示待ち / アプリ切替待ち / ダイアログ表示待ち / 操作対象の切り替え / テキスト表示待ち（OCR）
- クリックのアンカー基準（隅・中央）によるウィンドウサイズ追従
- 座標・OCR領域・ウィンドウクラスのクリック／ドラッグ取得
- グローバルショートカット、最前面ピン留め、実行ログ、ヘルプ（使い方・更新履歴）
- マクロのインポート／エクスポート（.vamacro）
- デザインシステムに沿った UI

### 安定性
- アプリ検出をプロセス名優先に変更し、複数ウィンドウ・タイトル変化に対応
- ウィンドウクラスを正規化照合（WPF の GUID、JUCE `JUCE_xxxx`、WinForms 等の起動毎の変化に追従）
- キー送信をスキャンコード送信化し、JUCE 製アプリでの複合キー（Ctrl+E 等）に対応
- 日本語 OCR の精度向上（明示的な日本語エンジン選択＋画像拡大）
- 座標取得時の異常終了（フック解放／ダイアログ状態）を修正
