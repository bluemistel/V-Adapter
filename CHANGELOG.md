# 更新履歴 / Changelog

本プロジェクトのすべての注目すべき変更を記録します。
形式は [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/)、バージョニングは [Semantic Versioning](https://semver.org/lang/ja/) に準拠します。

## [0.0.3] - 2026-06-27 (α)

### 追加
- **関連ソフトのランチャー機能**（合成音声ソフトや動画編集ソフトを本アプリから起動）
  - マクロ命令 **「対象アプリの起動」** を追加。対象アプリ管理に登録した実行ファイル、または命令で指定した exe を起動（起動失敗時はエラー表示、起動中ならスキップ可）。空欄時はスクリプトの対象アプリ、または送信先で選んだ対象アプリの実行ファイルを起動。
  - **対象アプリ管理に「実行ファイル（exe）」を登録**。各合成音声ソフトの標準インストール先を既定値として同梱（VOICEVOX はユーザー領域インストールのため空欄）。
  - **連携設定で環境ごとに動画編集ソフト（YMM4 / AviUtl / AviUtl2）の実行ファイルを登録**。
  - メイン画面ヘッダーに **電源ボタン** を追加し、現在の連携環境の動画編集ソフトを起動。
  - 組込マクロ **「合成音声ソフトの起動」** を同梱。
- **投げ込みの疎結合化**（内部アーキテクチャ。編集ソフトへの依存を本体から分離）
  - 編集ソフト非依存の中立契約 **DropPayload** と差し替え可能な投げ込みアダプタ **IImportAdapter** を導入。
  - AviUtl / AviUtl2 は組込 gcmz アダプタとして従来どおり動作（**挙動不変**）。
  - **「外部アダプタ（コマンド）」モード** を追加。`payload.json` を外部コマンドへ渡す参照実装で、有志が Premiere / DaVinci 等のアダプタをアウトオブツリーで作成・配布可能。
  - 仕様ドキュメント `docs/drop-payload-spec.md`（日英）を追加。
- **マクロの単体ファイル管理と完全ポータブル化**
  - 各マクロを **`.vamacro` 単体ファイル** として管理（`script/built-in` は同梱・更新で上書き、`script/user-script` はユーザー作成／インポートで温存）。
  - `library.json`（対象アプリ）・`integration.json`（連携設定）・`script/` を **exe 同階層に集約**（`%APPDATA%` 依存を解消）。初回起動時に旧データを自動移行。

### 変更
- 対象アプリ管理：起動時に先頭アプリを選択／「適用」を **「保存して閉じる」**（アクセント色）に変更／未保存時に取り消し確認ダイアログを表示。

## [0.0.2] - 2026-06-25 (α)

### 追加
- **AviUtl / AviUtl2 への音声・字幕 投げ込み連携**（ごちゃまぜドロップス外部連携API）
  - 連携環境を設定で3種から切替: マクロ動作ベース（YMM4 等）/ AviUtl+PSDToolKit / AviUtl2+PSDToolKit2。環境ごとに独立設定。
  - 出力フォルダを監視し、同名 wav+txt を検知（書き込み安定化・二重投入防止）→ 話者ルールでレイヤー解決 →
    ごちゃまぜドロップス（GCMZDrops / GCMZDrops2）へ `WM_COPYDATA` で投入。字幕生成は PSDToolKit に委譲。
  - **wav と txt を一緒に投入**（PSDToolKit の発動条件「同名 wav+txt を一緒にドロップ」に対応）。
  - **投入後シークをアイテム終端へ自動移動**（音声長 × プロジェクト fps を `frameAdvance` で指定、既定ON）。
  - 話者抽出の既定正規表現（`_話者_` / `-話者-`）をプリセット。
  - 「連携設定」画面：環境セレクタ、gcmz 接続/プロジェクト状態表示、監視フォルダ、既定レイヤー・シーク移動・
    安定待ち・margin(AviUtl2)、話者ルール表、ログ、「今すぐ投げる（テスト）」。
  - AviUtl / AviUtl2 両環境で動作確認。
- アプリ内「使い方」を **2ペイン式**に刷新（左メニュー＋右コンテンツ）。「AviUtl・AviUtl2 連携」「ライセンス・免責」を追加。

## [0.0.1] - 2026-06-23 (α)

初回リリース（α版）。

### 追加
- 標準マクロ「音声の再生（Ctrl+Space）」「音声の保存（Ctrl+E）」を同梱
- 対象アプリのテンプレート（VOICEVOX / A.I.VOICE2 / CeVIO AI / VOICEPEAK / VoiSona Talk）を同梱
- 1つのマクロに対象アプリごとのスクリプト（命令列）を保持する構造
- 実行時のスクリプト自動選択（前面アプリ判定）と「送信先」固定
- マクロ命令: 待機 / クリック / キー送信 / ウィンドウ表示待ち / アプリ切替待ち / ダイアログ表示待ち / 操作対象の切り替え / テキスト表示待ち（OCR）
- クリックのアンカー基準（隅・中央）によるウィンドウサイズ追従
- 座標・OCR領域・ウィンドウクラスのクリック／ドラッグ取得
- グローバルショートカット、最前面ピン留め、実行ログ、ヘルプ（使い方・更新履歴）
- マクロのインポート／エクスポート（.vamacro）
- デザインシステムに沿った UI

### 安定性
- アプリ検出をプロセス名優先に変更し、複数ウィンドウ・タイトル変化に対応
- ウィンドウクラスを正規化照合（WPF の GUID、JUCE `JUCE_xxxx`、WinForms 等の起動毎の変化に追従）
- キー送信をスキャンコード送信化し、JUCE 製アプリでの複合キー（Ctrl+E 等）に対応
- 日本語 OCR の精度向上（明示的な日本語エンジン選択＋画像拡大）
- 座標取得時の異常終了（フック解放／ダイアログ状態）を修正
